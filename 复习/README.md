# 复习系统 · 如何使用

> [← 返回目录](../README.md)

> [!NOTE]
> 4-5 个月的学习周期里，**前面学的容易忘**。本系统用 **Spaced Repetition + Active Recall + Bloom 分级** 保证知识留存。

## 2. 两种复习方式

### 方式 A · Anki（推荐给重度学习者）
1. 下载 [Anki](https://apps.ankiweb.net/) （免费）
2. `File → Import` 导入 `anki-import.csv`
3. 每天花 10-20 分钟过当日卡片
4. 按 Anki 的 SM-2 算法自动安排间隔

### 方式 B · 手动快读（推荐给轻度学习者）
按**"间隔递增"**手动过 `核心概念卡.md`：
- 第 1 天：学完新 Unit 后通读
- 第 3 天：合上答案自测
- 第 7 天：再自测一次
- 第 14 天
- 第 30 天
- 之后每月一次

**操作技巧**：建一个简单 markdown 表跟踪"今天该过哪些卡"——

```markdown
| 卡片范围 | 上次读 | 下次读 |
|---|---|---|
| Unit 0 (Q1-Q10) | 2026-05-15 | 2026-05-22 |
| Unit 1 (Q11-Q25) | 2026-05-19 | 2026-05-22 |
```

**或者**：直接把 `anki-import.csv` 喂给任何 SR 软件（Anki Desktop / AnkiDroid / Mochi / RemNote / Apple Shortcuts + ChatGPT），让算法替你排。**强烈不推荐用 Excel 排间隔**——会忘。

**没坚持的成本**：4 个月后 Unit 0/1 基本忘光。

## 3. Active Recall 提问的用法

和 SR 卡不同——这些是**开放式问题**，用来练"**自己把东西讲清楚**"。

建议用法：

1. **每学完一章，拿其中 2-3 道题**
2. **合上书，口头或书面回答**
3. 回答完**再翻书对比**
4. 记下"没想到 / 讲错了"的点
5. 可以贴给 AI：**"挑我这个回答的漏洞"**（但别让它给答案）

---

## 4. Bloom 分级标注的用法

本书的主要练习都标了 Bloom 六级之一：

```
记忆 → 理解 → 应用 → 分析 → 综合 → 评估
```

- 如果你发现自己**一直停在"理解"**（看懂但不会用）——该做更多 **应用 / 分析** 类练习
- 如果你**跳过了"应用"直接做"综合"**——会产出"看起来完整但漏洞百出"的方案
- 详细见 [Bloom分级说明](Bloom分级说明.md)

## 5. 复习节奏推荐

| 时间 | 动作 | 预计耗时 |
|---|---|---|
| 学完一个 Unit | 通读当 Unit 对应的 SR 卡 | 15 分钟 |
| 学完一个 Unit | 回答 2-3 道 Active Recall | 20-30 分钟 |
| 每周 | Anki 新卡 + 复习（自动调度）| 10-15 分钟/天 |
| 每月末 | 回答 [附录 A · 自检表](../附录/A-每月自检表.md) + 抽 5 张老卡自测 | 20 分钟 |

---

## 6. 为什么这套东西重要

| 学习科学原理 | 本系统的应用 |
|---|---|
| **Spaced Repetition**（Ebbinghaus）| 对抗遗忘曲线，按间隔复习 |
| **Active Recall / Testing Effect** | 从记忆中提取比重读有效 3-5× |
| **Elaboration** | Active Recall 题强制你"关联已有知识" |
| **Bloom's Taxonomy** | 让你看到自己的学习停留在哪一层，避免"看似懂了" |
| **Metacognition** | 暴露"没想到"的盲区，是最重要的学习信号 |

---

## 7. 给 SRE 的一句话提醒

> [!IMPORTANT]
> **学完不复习 = 没学**。
>
> 4-5 个月跨度上，**没有 SR 机制的自学者平均 30 天后遗忘 60%**（Ebbinghaus curve 依然有效）。
>
> 这个复习系统不是"加分项"，**是让前 3 个月没白学的保险**。
